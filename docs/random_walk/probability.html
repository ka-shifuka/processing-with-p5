Penjelasan


Probabilitas pada random() sangat sederhana buat sebuah perbandingan seperti dibawah ini 

if (random() < 0.5) { do something }

Yang artinya dengan 50% kemunkinan lakukan sesuatu


Fungsi random() pada p5js mengembalikan float number dari 0 dan 1. Dengan membuat perbandingan pada fungsi random() kita bisa membuat probabilitas dengan menentukan seberapa besar peluang terjadi jika random() < 0.1 artinya 10% dan kalian bisa bermain main dengan probabilitas ini.
